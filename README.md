# --Open Pokedex--
![image alt](https://github.com/LorenWedderburn/Pokedex/blob/7998a0788fce46184f21ca40036bbf4ea3d3f3c0/PokeOpen.JPG)
Once open the Pokedex offers you several options. 
### On the right hand side there are modules. 
1. Check the weather in any city globally
2. Check the time
3. Add text notes for any pins dropped on the map
### On the left hand side
1. Dex allows you to switch to the screen mode that allows you to search for different Pokemon
2. Map allows you to bring up a map of your local area and drop pins of the pokemon you encounter
3. Search allows you to enter a name of a pokemon into the text box, display an image of the pokemon and its stats
4. Select allows you yo lock in a pokemon you have searched and allows you to drop their specific encounter pin on the map
